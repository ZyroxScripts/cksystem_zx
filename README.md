# cksystem_zx
ZX Character Kill System for FiveM - allows admins to delete player characters with security confirmation. Supports ESX &amp; QB-Core, multilingual (CS/EN/DE/ES/FR), ox_lib dialogs &amp; configurable permissions. Includes /ck and /ckinsta commands for different admin levels.
